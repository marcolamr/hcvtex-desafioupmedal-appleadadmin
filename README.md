# Hiring Coders #2 - Desafio UpMedal - App Customizado Captura de Leads Admin

App Admin desenvolvido no padrão Vtex IO, como parte do treinamento Hiring Coders, promovido pela [Vtex](https://vtex.com/br-pt/) e realizado pela [Gama Academy](https://www.gama.academy/).

## Funcionalidade

Bloco Vtex IO para ser usado no Admin que lista os leads capturados através do [App Store Front](https://github.com/marcolamr/hcvtex-desafioupmedal-applead), integrado com AWS Api Gateway e busca dos dados persistidos no DynamoDB.
